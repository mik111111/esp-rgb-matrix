# Led 32x32 Matrix 
Features:
* Web interface
* REST API
* Low level pixel UDP API
* Python library

Components:
* ESP 8266 (any board, optionaly with small OLED screen for debugging)
* Pixel booster (3.3V -> 5V)
* 32x32 WS2812B panel (may be combined with smaller panels connected in serie)

Sofware:
* Arduino IDE
* Python
